# CONVERSORDEUNIDADE
The project consists of a unit conversion calculator implemented in HTML and JavaScript. The aim is to provide users with a simple and practical tool for performing conversions between different measurement categories, such as length, weight, and temperature.

In the HTML, elements such as category selectors, source and destination units, input fields for the values to be converted, and a result field to display the conversion result were created. The layout was kept clean and intuitive, facilitating understanding and usage by users.

In the JavaScript, logic was implemented to dynamically update the options for source and destination units based on the category selected by the user. This allows the options to be customized according to the chosen category, making it easier to select the desired units of measurement.

Additionally, a specific function was created to convert meters to centimeters. This function takes the value in meters entered by the user, performs the necessary calculation, and displays the result in centimeters in the designated field. This functionality serves as an example of implementing a specific conversion and can be expanded to include other conversions as needed.

The choices made prioritized simplicity and ease of use for the user. Intuitive options such as category and unit selectors were chosen, which are easy to understand and use. Furthermore, the code was organized in a clear and modular manner, allowing for easy expansion to include more categories and units of measurement if desired.

The project was successfully delivered and is functioning correctly. It provides a practical solution for performing unit conversions quickly and efficiently. Based on the project's needs and requirements, decisions were made to provide a good user experience while ensuring flexibility for future expansions and improvements.

O projeto consiste em uma calculadora de conversão de unidades de medida implementada em HTML e JavaScript. A ideia é fornecer aos usuários uma ferramenta simples e prática para realizar conversões entre diferentes categorias de medidas, como comprimento, peso e temperatura.

No HTML, foram criados elementos como seletores de categoria, unidades de origem e destino, bem como campos de entrada para os valores a serem convertidos e um campo de resultado para exibir o resultado da conversão. O layout foi mantido limpo e intuitivo, facilitando a compreensão e utilização por parte dos usuários.

No JavaScript, foi implementada a lógica para atualizar dinamicamente as opções de unidades de origem e destino, com base na categoria selecionada pelo usuário. Isso permite que as opções sejam personalizadas de acordo com a categoria escolhida, facilitando a seleção das unidades de medida desejadas.

Além disso, uma função específica foi criada para converter metros em centímetros. Essa função recebe o valor em metros digitado pelo usuário, realiza o cálculo necessário e exibe o resultado em centímetros no campo designado. Essa funcionalidade serve como um exemplo da implementação de uma conversão específica, podendo ser expandida para incluir outras conversões conforme necessário.

As escolhas feitas priorizaram a simplicidade e facilidade de uso para o usuário. Optou-se por apresentar opções intuitivas, como seletores de categoria e unidades, que são fáceis de entender e utilizar. Além disso, o código foi organizado de maneira clara e modular, permitindo uma fácil expansão para incluir mais categorias e unidades de medida, caso seja desejado.

O projeto foi entregue com sucesso e está funcionando corretamente. Ele oferece uma solução prática para realizar conversões de unidades de medida de forma rápida e eficiente. Com base nas necessidades e requisitos do projeto, foram tomadas decisões que visam proporcionar uma boa experiência ao usuário, além de garantir a flexibilidade para futuras expansões e melhorias.
